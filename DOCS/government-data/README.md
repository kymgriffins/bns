# Government Data — Document Summaries & Data Points

This directory holds simplified summaries of official Kenyan government budget documents.  
Each document has its own folder named by document type and fiscal year.

## Structure

```
government-data/
├── README.md                    ← this file
├── national/
│   ├── BPS-2025-26/             ← Budget Policy Statement
│   ├── ESTIMATES-2025-26/       ← Estimates of Revenue & Expenditure
│   ├── FINANCE-BILL-2025/       ← Finance Bill / Act
│   └── APPROPRIATION-2025/      ← Appropriation Bill / Act
├── county/
│   ├── NAIROBI-2025-26/
│   ├── MOMBASA-2025-26/
│   └── ...
└── oversight/
    └── AUDIT-HIGHLIGHTS-2025/
```

## How to Add a Document

1. Create a folder: `[SCOPE]/[DOC-TYPE]-[FY]/`
2. Drop in the summary file (PDF, markdown, or txt — any format)
3. Create a companion `data-points.md` in the same folder using the template below

---
