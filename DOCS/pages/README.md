# Pages — Progress Tracker

Progress, todos, and work logs for the three main public pages.  
Each file is organized by date entries. Add a new `## YYYY-MM-DD` entry each session.

## Files

| Page | File | Status |
|------|------|--------|
| `/` Home landing | [landing.md](./landing.md) | 🟡 Functional — needs trim + real data |
| `/about` | [about.md](./about.md) | 🔴 Half-built — dead code, no real people |
| `/reports` | [reports.md](./reports.md) | 🔴 Critical — no reports rendered |

## Workflow

1. Pick a page file
2. Add a new date entry under `## Sessions`
3. Check off todos as you go
4. Log what you did in the progress log at the bottom
